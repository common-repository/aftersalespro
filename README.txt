== Frequently Asked Questions ==

= Do I need to install another software to use AfterSalesPro plugin =
No. You just need to install and set up the plugin. Then you can have full access to all services through a SaaS administration by using your account.

= Do I have to pay to create the AfterSalesPro account? =
No. You do not need to pay anything for your account creation. You can start using the service for free (Free plan available). For advanced features you can then choose the plan that fits your needs best.

= How many courier companies can I use simultaneously? =
You can use as many as you like with no additional charges. Any fee for paid plans is related to the number of shipments and not the number of courier companies you use.

= Can I create multiple vouchers at the same time =
Yes. You can massively create and print vouchers with only a few clicks
